### SessionSetup.js dark overrides

renderAttendance() / renderAttendee() (member rows):
- Row label: `bg-white rounded-xl border border-gray-100 shadow-sm` → add `dark:bg-gray-800 dark:border-gray-700`
- Member name span: `text-lg font-medium` → add `dark:text-gray-100`
- Last played sub-text: `text-[10px] text-gray-400 uppercase font-bold` → add `dark:text-gray-500`
- Never played sub-text: `text-[10px] text-gray-300 uppercase font-bold` → add `dark:text-gray-500`

Static template (el.innerHTML):
- Hint text `text-gray-500`: add `dark:text-gray-400`
- Invert button `text-blue-600 font-bold`: add `dark:text-blue-400`
- Sticky start button bar `bg-gray-50/90 border-t border-gray-100`: add `dark:bg-gray-800/90 dark:border-gray-700`

### ClubManager.js dark overrides

renderClubs() function:
- Club card container (ternary): `bg-white ... border ${isCurrent ? 'border-blue-500 bg-blue-50/30' : 'border-gray-100'}`
  → base: add `dark:bg-gray-800`
  → active branch: add `dark:border-blue-400 dark:bg-blue-900/20`
  → inactive branch: add `dark:border-gray-700`
- Member count text `text-gray-500`: add `dark:text-gray-400`
- Club name: wherever `font-bold text-lg` is the club name → add `dark:text-gray-100`
- Delete button `text-gray-400 hover:text-red-600`: add `dark:text-gray-500 dark:hover:text-red-400`
- Empty state card `bg-white border-2 border-dashed border-gray-200`: add `dark:bg-gray-800 dark:border-gray-600`
- Empty state text `text-gray-500` / `text-gray-400`: add `dark:text-gray-400` / `dark:text-gray-500`

Static template:
- New club form `bg-blue-50 border border-blue-100`: add `dark:bg-blue-900/20 dark:border-blue-800`
- New club input (if present): add `dark:bg-gray-700 dark:border-gray-600 dark:text-gray-100`

### MemberEditor.js dark overrides

renderMembers() function:
- Member row div `bg-white border border-gray-100`: add `dark:bg-gray-800 dark:border-gray-700`
- Member name span `font-medium text-lg`: add `dark:text-gray-100`
- Edit/Remove action buttons `text-gray-400`: add `dark:text-gray-500`
- Empty state card `bg-white border-2 border-dashed border-gray-200`: add `dark:bg-gray-800 dark:border-gray-600`
- Empty state text `text-gray-500 italic`: add `dark:text-gray-400`

Static template (el.innerHTML):
- Start Session card `bg-blue-50 border border-blue-100`: add `dark:bg-blue-900/20 dark:border-blue-800`
- Start Session heading `text-blue-800`: add `dark:text-blue-300`
- Start Session sub-text `text-blue-600`: add `dark:text-blue-400`
- Roster heading `text-gray-700 uppercase`: add `dark:text-gray-400`
- Name input `bg-white border border-gray-200` (or border-gray-300): add `dark:bg-gray-700 dark:border-gray-600 dark:text-gray-100`

### Help.js dark overrides (static template):
- Back button `text-blue-600`: add `dark:text-blue-400`
- Section heading `text-blue-600` or `text-gray-900 font-bold`: add `dark:text-blue-400` or `dark:text-gray-100`
- Section icon badge `bg-blue-100 text-blue-600 rounded-full`: add `dark:bg-blue-900/30 dark:text-blue-400`
- Content card `bg-white border border-gray-100 shadow-sm`: add `dark:bg-gray-800 dark:border-gray-700`
- Content card body text (add `dark:text-gray-300` to the card container for inherited coverage, or add explicitly to `text-sm leading-relaxed` paragraphs)
- Hint/aside text `text-gray-500`: add `dark:text-gray-400`
- Footer/closing text `text-gray-400`: add `dark:text-gray-500`
